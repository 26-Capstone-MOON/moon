# Domain Rules

## Decision Point (DP)
- DP = a point where user must make a navigation decision
- Types: turn, crosswalk, stairs/ramp, entrance/exit, confirmation
- Extracted from Tmap API by turnType code
- Frontend receives DPs from backend, does NOT create them

### turnType Mapping
| turnType | Meaning | Category |
|---|---|---|
| 12~19 | turn / U-turn | direction_change |
| 211~217 | crosswalk | crosswalk |
| 127~129, 218 | stairs / ramp / elevator | vertical |
| 125~126 | overpass / underpass | infrastructure |
| 200, 201 | origin / destination | origin / destination |
| 11 | straight | → Virtual DP |
| 1~7 | no guidance | excluded |

## Virtual DP
- Inserted on long straight segments (~200m intervals)
- Purpose: confirmation, not direction change
- isVirtual: true, category: "confirmation"
- Example: "GS25를 지나 계속 직진하세요"

## Landmark
- Selected by backend scoring model
- Frontend NEVER selects landmarks
- Frontend receives: name, position (left/right/front), matchStatus
- If no landmark → fallback to basic instruction

## Guidance Text
- Generated by backend LLM
- Frontend only: displays text, plays TTS, triggers vibration
- Examples:
  - "투썸플레이스에서 우회전 후 직진"
  - "횡단보도를 건넌 후 직진하세요"
  - "이 길을 따라 계속 직진하세요"

## Panorama
- Street view image of the DP location
- Supplementary — user sees it on screen, not forced
- Backend provides panoramaUrl in DecisionPoint data
- If unavailable or old (>2 years) → skip or show warning

## Off-Route
- Detected by: sustained distance from polyline + heading divergence
- Single GPS spike = NOT off-route
- Frontend: show red overlay alert, request re-route via API
- Backend: recalculates route + new DPs + new guidance

## Scoring Model (Backend Reference)
Frontend does not implement this. For understanding only:
- Landmarks scored by: visual salience × semantic salience × trust
- Trust: MATCHED (1.0) > POI_ONLY (0.3) > VISION_ONLY (0.1)
- Environment affects scores: time, weather, season, business hours
- LLM writes sentence but does NOT choose landmark

## Fallback Rule
Always have fallback guidance.
If no landmark → use direction + distance only.
Never show empty guidance.
